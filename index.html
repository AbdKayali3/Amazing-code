How to use

  1. Paste your code in the text area
  2. Choose the language, the theme, and set other options if needed
  3. The tool will do its magic and show you the styled code
  4. Click Copy HTML button to use it in your blog/email/homework or any texteditor
  5. Click on Copy RichText to use it for Word documents like MS Word or Google Docs
